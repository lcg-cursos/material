La función print() en Python

La función print() es una de las funciones más básicas y utilizadas en Python. Su propósito es mostrar información en la consola. Es útil para depuración, mostrar resultados de cálculos y mejorar la interacción con el usuario.

1. Uso Básico de print()

La sintaxis más simple de print() consiste en escribir la función seguida de los valores que queremos mostrar entre paréntesis:

print("Hola, mundo!")

También se pueden imprimir números y otros tipos de datos:

print(42)
print(3.1416)
print(True)

2. print() con Múltiples Argumentos

Podemos imprimir varios valores separándolos con comas. Python agregará automáticamente un espacio entre ellos.

nombre = "Ana"
edad = 25
print("Nombre:", nombre, "Edad:", edad)

3. Uso del Parámetro sep

El parámetro sep permite especificar cómo separar los valores cuando se imprimen múltiples argumentos.

print("A", "B", "C", sep="-")
print("Python", "es", "genial", sep=" * ")

4. Uso del Parámetro end

El parámetro end define qué debe imprimirse al final de la salida en lugar del salto de línea por defecto.

print("Hola", end=" ")
print("Mundo!")

Otro ejemplo:

print("Procesando...", end="")
print("Listo!")

5. print() con f-strings (Cadenas Formateadas)

Desde Python 3.6, se introdujeron las cadenas formateadas (f-strings), que permiten insertar variables dentro de una cadena de manera más legible.

nombre = "Carlos"
edad = 30
print(f"Mi nombre es {nombre} y tengo {edad} años.")

También se pueden hacer cálculos dentro de f-strings:

a = 5
b = 3
print(f"La suma de {a} y {b} es {a + b}.")

Se pueden aplicar formatos a los valores, por ejemplo, para números decimales:

pi = 3.1415926535
print(f"El valor de pi con dos decimales es: {pi:.2f}")

6. Imprimir Caracteres Especiales

Podemos usar caracteres de escape para dar formato al texto:

  • \n → Salto de línea
  • \t → Tabulación
print("Primera línea\nSegunda línea")
print("Columna 1\tColumna 2")
