Condicionales en Programación

Introducción

Los condicionales son estructuras fundamentales en programación que permiten tomar decisiones en función de ciertas condiciones. Esto significa que un programa puede ejecutar diferentes bloques de código dependiendo de si una condición es verdadera o falsa.

Concepto Básico

Un condicional evalúa una expresión lógica (que devuelve True o False) y ejecuta un bloque de código dependiendo del resultado.

Sintaxis en Python

En Python, el condicional más común es if, que se usa junto con elif (opcional) y else (opcional) para manejar múltiples casos.

if condicion:
    # Código a ejecutar si la condición es verdadera
elif otra_condicion:
    # Código a ejecutar si la otra condición es verdadera
else:
    # Código a ejecutar si ninguna condición es verdadera

Condicional if

Ejemplo básico:

edad = 18
if edad >= 18:
    print("Eres mayor de edad")

🔹 Explicación: Si la variable edad es mayor o igual a 18, se ejecutará la línea print("Eres mayor de edad"). Si edad fuera menor a 18, no se ejecutaría nada.

Condicional if con else

Si queremos ejecutar una acción cuando la condición no se cumple, usamos else.

edad = 16
if edad >= 18:
    print("Eres mayor de edad")
else:
    print("Eres menor de edad")

🔹 Explicación: Si edad es menor a 18, se ejecuta la parte del else.

Condicional if con elif

elif (abreviatura de “else if”) permite evaluar múltiples condiciones.

temperatura = 25

if temperatura > 30:
    print("Hace mucho calor")
elif temperatura > 20:
    print("El clima es agradable")
else:
    print("Hace frío")

🔹 Explicación:

  • Si la temperatura es mayor a 30, imprime “Hace mucho calor”.
  • Si la temperatura está entre 21 y 30, imprime “El clima es agradable”.
  • Si la temperatura es menor o igual a 20, imprime “Hace frío”.

Condicionales Anidados

Podemos colocar condicionales dentro de otros para evaluar más de una condición.

edad = 20
tiene_identificacion = True

if edad >= 18:
    if tiene_identificacion:
        print("Puedes entrar al evento")
    else:
        print("Necesitas una identificación")
else:
    print("No puedes entrar al evento")

🔹 Explicación: Solo se permite la entrada si la persona es mayor de 18 y tiene identificación.

Operadores Lógicos en Condicionales

Podemos combinar condiciones con operadores lógicos (and, or, not).

Los operadores lógicos permiten combinar condiciones y evaluar expresiones booleanas (True o False). Las tablas de verdad son una herramienta que nos ayuda a entender el comportamiento de estos operadores.

A B NOT A A AND B A OR B A XOR B (A ≠ B)
True True False True True False
True False False False True True
False True True False True True
False False True False False False 
usuario_activo = True
tiene_permisos = False

if usuario_activo and tiene_permisos:
    print("Acceso concedido")
else:
    print("Acceso denegado")

operadores en Python que se utilizan con frecuencia en genómica y bioinformática, especialmente en condicionales (if, elif, else).

🔹 1. Operadores de Comparación (==, !=, <, >, <=, >=)

Se usan para comparar valores y evaluar condiciones.

Operador Descripción Ejemplo en Bioinformática
== Igual a if secuencia == "ATGC": (Verifica si la secuencia es "ATGC")
!= Distinto de if mutacion != "C>T": (Verifica si la mutación no es "C>T")
< Menor que if longitud < 500: (Evalúa si la longitud de la secuencia es menor a 500)
> Mayor que if expresion > 10: (Evalúa si la expresión génica es mayor a 10)
<= Menor o igual que if score <= 0.05: (Evalúa si el valor score es significativo)
>= Mayor o igual que if cobertura >= 30: (Verifica si la cobertura de secuenciación es suficiente)

🔹 2. Operadores Lógicos (and, or, not)

Se utilizan para combinar condiciones en análisis complejos.

Operador Descripción Ejemplo en Bioinformática
and Ambas condiciones deben ser True if gc_porcentaje > 40 and gc_porcentaje < 60: (Evalúa si el contenido GC está dentro del rango óptimo)
or Al menos una condición debe ser True if "N" in secuencia or "?" in secuencia: (Verifica si la secuencia contiene caracteres ambiguos)
not Invierte la condición if not mutacion_encontrada: (Ejecuta el código solo si no se encuentra la mutación)

🔹 3. Operadores de Pertenencia (in, not in)

Se utilizan para verificar si un elemento está presente en una secuencia, lista o conjunto.

Operador Descripción Ejemplo en Bioinformática
in Verifica si un elemento está presente if "ATG" in secuencia: (Verifica si la secuencia contiene el codón de inicio ATG)
not in Verifica si un elemento no está presente if "N" not in secuencia: (Evalúa si no hay bases ambiguas en la secuencia)

🔹 4. Operadores de Identidad (is, is not)

Se usan para comparar si dos variables hacen referencia al mismo objeto en memoria.

Operador Descripción Ejemplo en Bioinformática
is Verifica si dos variables son el mismo objeto if resultado is None: (Verifica si resultado es None)
is not Verifica si dos variables son diferentes objetos if secuencia is not otra_secuencia: (Evalúa si las secuencias no son la misma instancia)
