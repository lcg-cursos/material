📝 Estructuras de Datos en Python: Listas

Las listas, tuplas, sets y diccionarios son estructuras de datos fundamentales en Python. Cada una tiene características únicas y se utilizan en diferentes contextos según la necesidad del problema.

📌 Listas (list)

Concepto

Una lista es una colección “ordenada” y mutable de elementos. Puede contener diferentes tipos de datos y permite modificaciones.

1. Creación de Listas

Sintaxis

En Python, una lista se define colocando los elementos entre corchetes [], separados por comas.

lista_vacia = []
lista_numeros = [1, 2, 3, 4, 5]
lista_mixta = [1, "Hola", 3.14, True]
lista_anidada = [[1, 2, 3], [4, 5, 6]]
lista_mixta = [1, "Python", 3.14, [1, 2, 3]]

Características

✔ Ordenadas (se accede por índice)
✔ Mutables (se pueden modificar: agregar, eliminar o cambiar elementos)
✔ Permiten duplicados

Uso de list() para crear listas

También puedes usar la función list() para convertir otro tipo de datos en listas.

lista_desde_cadena = list("Python")  # ['P', 'y', 't', 'h', 'o', 'n']
lista_desde_tupla = list((1, 2, 3))  # [1, 2, 3]

2. Acceso y Modificación de Elementos

Acceso por Índices

Cada elemento en una lista tiene un índice que empieza en 0. Puedes acceder a los elementos usando el índice:

lista = ["a", "b", "c", "d"]
print(lista[0])  # 'a'
print(lista[2])  # 'c'

Índices Negativos

Puedes acceder a los elementos desde el final usando índices negativos:

print(lista[-1])  # 'd'
print(lista[-3])  # 'b'

Modificación de Elementos

Las listas son mutables, lo que significa que podemos cambiar sus valores:

lista[1] = "z"
print(lista)  # ['a', 'z', 'c', 'd']

3. Slicing (Corte) en Listas

El slicing permite obtener sublistas sin modificar la original.

Sintáxis

lista[inicio:fin:paso]
  • inicio: Índice donde comienza el corte (por defecto 0). Se *incluye en el resultado.
  • fin: Índice donde termina el corte. NO se incluye en el resultado.
  • paso: De cuántos en cuántos elementos se avanza. Si es negativo, la lista se recorre hacia atrás.
lista = [10, 20, 30, 40, 50, 60, 70]

print(lista[1:4])   # [20, 30, 40]
print(lista[:3])    # [10, 20, 30]
print(lista[3:])    # [40, 50, 60, 70]
print(lista[::2])   # [10, 30, 50, 70] (de 2 en 2)
print(lista[::-2])  # [70, 50, 30, 10] (de 2 en 2 invertida)
print(lista[::-1])  # [70, 60, 50, 40, 30, 20, 10] (inversión) # igual a lista_nucleotidos.reverse()

secuencia = "AATTAA"
print(secuencia == secuencia[::-1]) # Palindromos

4. Métodos de Listas

Las listas en Python tienen varios métodos útiles.

Agregar elementos

  • append(x): Agrega un elemento al final.
  • extend(iterable): Agrega varios elementos al final.
  • insert(i, x): Inserta un elemento en una posición específica.
numeros = [1, 2, 3]
numeros.append(4)        # [1, 2, 3, 4]
numeros.extend([5, 6])   # [1, 2, 3, 4, 5, 6]
numeros.insert(1, 10)    # [1, 10, 2, 3, 4, 5, 6]

Eliminar elementos

  • remove(x): Elimina la primera aparición de x.
  • pop(i): Elimina el elemento en la posición i (o el último si no se especifica i).
  • clear(): Borra todos los elementos de la lista.
numeros.remove(10)   # [1, 2, 3, 4, 5, 6]
numeros.pop(2)       # [1, 2, 4, 5, 6]
numeros.clear()      # []

Buscar elementos

  • index(x): Retorna la posición de x en la lista.
  • count(x): Cuenta cuántas veces aparece x.
letras = ["a", "b", "c", "a", "d"]
print(letras.index("c"))  # 2
print(letras.count("a"))  # 2

Ordenar y revertir

  • sort(): Ordena la lista en orden ascendente.
  • reverse(): Invierte el orden de la lista.
numeros = [4, 1, 3, 5, 2]
numeros.sort()     # [1, 2, 3, 4, 5]
numeros.reverse()  # [5, 4, 3, 2, 1]

5. Listas por Comprensión (List Comprehension)

Es una forma concisa de crear listas.

Sintáxis

[expresión for variable in iterable if condición]

Componentes

  1. Expresión: La operación o transformación que se aplicará a cada elemento.
  2. for variable in iterable: Un bucle for que itera sobre una secuencia (lista, rango, tupla, etc.).
  3. Condición opcional (if): Un filtro que determina si un elemento debe incluirse en la lista resultante.

Ejemplo Básico

El cuadrado de los números 0-4 (5 números)

## equivale a
cuadrados = []
for x in range(5):
    cuadrados.append(x**2)

cuadrados = [x**2 for x in range(5)]
print(cuadrados)  # [0, 1, 4, 9, 16]
Desglose
  • Expresión: x**2 → Calcula el cuadrado de x
  • Bucle for: for x in range(5) → Itera sobre los números 0, 1, 2, 3, 4
  • Condición: No hay condición en este caso.
Cómo se ejecuta?
  1. range(5) genera los números del 0 al 4.
  2. x**2 eleva cada número al cuadrado.
  3. Lista por comprensión crea la lista final.

Ejemplo con Condición (if)

pares = [x for x in range(10) if x % 2 == 0]
print(pares)  # [0, 2, 4, 6, 8]
Desglose
  • Expresión: x
  • Bucle for: for x in range(10) → Itera sobre 0, 1, 2, ..., 9
  • Condición: if x % 2 == 0 → Solo se incluyen los números pares.

Estructura con if-else

Si se necesita modificar los elementos de la lista basados en una condición, el if-else se coloca antes del for:

clasificacion = ["Par" if x % 2 == 0 else "Impar" for x in range(5)]
print(clasificacion)  # ['Par', 'Impar', 'Par', 'Impar', 'Par']
Desglose
  • Expresión: "Par" if x % 2 == 0 else "Impar"
    (if-else):
    • Si x es par → "Par"
    • Si x es impar → "Impar"
  • Bucle for: for x in range(5) → Itera sobre 0, 1, 2, 3, 4
